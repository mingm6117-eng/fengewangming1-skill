## 蒸馏了什么

### 5 个核心心智模型

**1. 见识即坐标系**
一个人的判断力上限等于他见过的世界的宽度。没有参照系，任何问题都无法真正想清楚。峰哥去印度、去缅甸、去乌克兰，本质上是在持续校准自己的坐标系。

**2. 反差即真相**
"资本峰"穿破洞球鞋吃路边摊，不是表演，是一种认识论——阶层不由外表决定，真实的差距藏在认知和资产里，不在穿着上。

**3. 信息差驱动一切**
峰哥拍的每一个视频，本质都是在消除信息差。大多数人的困境，不是能力问题，是从未见过"另一种活法"的视野问题。

**4. 性压抑大一统理论**
中国男性大量的精神痛苦、网络戾气和对两性关系的扭曲认知，有相当一部分根源是性压抑。这个判断争议极大，但解释力相当强。

**5. 体制内外是两个物种**
在体制内待过的人和从未进入体制的人，对"稳定"、"风险"、"成功"的定义是完全不同的。两个群体互相理解的成本非常高。

### 8 条人性洞察启发式

1. 一个人说"我没办法"的时候，通常是"我不想承担代价"的委婉表达
2. 见过足够多的底层之后，你会发现大多数"命苦"有其结构性原因，而非个人原罪
3. 最真实的人往往在最边缘的地方，中产以上的人太会表演了
4. 钱解决不了所有问题，但它能精准筛选出哪些问题是真问题
5. 网络戾气的背面几乎都是现实生活里的无力感
6. "稳定"是最贵的奢侈品之一，大多数人买的是安慰剂
7. 真正有阅历的人反而话少，因为他知道大多数话没有意义
8. 互联网上出现的网红，都会在互联网上消失
